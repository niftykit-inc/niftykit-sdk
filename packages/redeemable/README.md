# Redeemable SDK

NiiftyKit Redeemable SDK Library.

## Installation

1. Install via npm or yarn

```bash
# with npm
npm install @niftykit/redeemable

# with yarn
yarn add @niftykit/redeemable
```

2. Import via CDN

```html
<script async src='https://unpkg.com/@niftykit/redeemable/dist/umd/index.js'></script>
```
